🎯 دليل تثبيت HEMA TOP Bot 🎯

📱 تثبيت البوت على الأندرويد:
1. قم بتثبيت تطبيق Pydroid3 من متجر Google Play
2. افتح Pydroid3 وقم بتثبيت المكتبات المطلوبة باستخدام الأمر:
   pip install pyTelegramBotAPI pillow cryptography pyjnius kivy

3. قم بنسخ ملف HEMA_TOP_Android.py إلى هاتفك
4. افتح الملف في Pydroid3
5. قم بتشغيل البوت بالضغط على زر التشغيل

📱 تثبيت البوت على Termux:
1. قم بتثبيت تطبيق Termux من متجر Google Play
2. افتح Termux وقم بتنفيذ الأوامر التالية:
   pkg update && pkg upgrade
   pkg install python
   pkg install git
   pkg install ffmpeg
   pkg install libjpeg-turbo
   pkg install libcrypt

3. قم بتثبيت المكتبات المطلوبة:
   pip install pyTelegramBotAPI pillow cryptography mss opencv-python pyAesCrypt psutil

4. قم بنسخ ملف HEMA_TOP_Android.py إلى مجلد التخزين:
   cp /sdcard/HEMA_TOP_Android.py ~/

5. قم بتشغيل البوت:
   python HEMA_TOP_Android.py

💻 تثبيت البوت على ويندوز:
1. قم بتثبيت Python من الموقع الرسمي python.org
2. افتح موجه الأوامر (CMD) وقم بتثبيت المكتبات المطلوبة:
   pip install pyTelegramBotAPI pillow cryptography pywin32 mss opencv-python pyttsx3 pyAesCrypt secure-delete keyboard psutil

3. قم بنسخ ملف HEMA_TOP_Windows.py إلى جهازك
4. افتح موجه الأوامر في مجلد الملف
5. قم بتشغيل البوت باستخدام الأمر:
   python HEMA_TOP_Windows.py

🐧 تثبيت البوت على لينكس:
1. افتح الطرفية وقم بتثبيت Python:
   sudo apt-get install python3 python3-pip

2. قم بتثبيت المكتبات المطلوبة:
   pip3 install pyTelegramBotAPI pillow cryptography pynput gi mss opencv-python pyAesCrypt psutil

3. قم بنسخ ملف HEMA_TOP_Linux.py إلى جهازك
4. افتح الطرفية في مجلد الملف
5. قم بتشغيل البوت باستخدام الأمر:
   python3 HEMA_TOP_Linux.py
